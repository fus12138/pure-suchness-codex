§3 纯质自动：从自呈到自显的必然性

显现既已确立，其根据归于纯质之自呈。然而，自呈之位并非静止之端点，而是动力之起点。此刻理性必须追问：自呈者之存在方式为何？若其仅为静态之在，则“明性”如何成其为“使显之能”？若其必为能动之在，则此“动”的必然根据何在？

§3.1 “永恒静寂”假设的先验检验

设定纯质为永恒静寂，意即其自呈恒守无差别之同一。此设定须受理性之检验。

（一）能之虚设
明性之规定在于“使之显”。若纯质永无“使之显”的现实，则“能显”沦为无可实现之空名；无可实现之“能”在先验上等同于“无能”。与纯质作为“明性之源”的起断相冲。

（二）思之自反
当下思辨本身是一显现之事：必预设“能思/所思”的位差与开敞。若纯质绝对静寂而无任何可分化之可能，则思辨发生之全部先验条件被一并取消。承认此假设，即是否定此刻思辨正在发生之事实，自陷矛盾。

小结： “永恒静寂”的纯质取消“能显”的可实现性，并抹除思之现实性，故为先验不可能。

§3.2 自显的必然：同一之双向式

既然永恒静寂被排除，为维持理性自洽与思之现实，必断定：
纯质之自呈，逻辑地蕴含自显的必然。

“自显”指纯质必然实现其“使之显”功能之先验规定。

  • 自呈： 其本体之位（它是明的）；
  • 自显： 其功能之实（它必使之明）。

二者非时间序列，而为一体两面：其“在”即其“显”，其“是”即“使之是”。

此“动”非经验学变更，而是先验之实现之势：纯质作为明性源头之逻辑完成。 由此可得先验动力学第一原理：
纯质之必然自动——纯质之“在”与“显”同一，故其自呈必自显；静寂之纯质乃理性悖论，能动方为其作为显现终极根据之必要位格。
此动非时间性的常在性（即不可止），将见于下一节之证明。

§3.3 推论与过渡

推论1（活动非外加）： 此“动”为本源自启，非外物所激。 若此实现仍需他因触发，则他因居先，纯质不复为源；理性退入无限后退，立刻自毁。
推论2（实现性与非时序性）： 纯质之“自显”，非时间中的过程，而是其存在之实现方式。若其“自呈”之在不能同时即是“自显”之实现，则“自呈”将沦为永不兑现的空洞承诺，其“明性”位格亦将落空。故，此“动”是本体论上的实现性，而非宇宙论上的过程性；是“现成—使成”的绝对同时，不倚赖任何先后序列。
推论3（内在极化之必然）： 自显若要避免其“显”落空，则必须实现为可辨之相。然此“可辨之相”不能从外引入，其唯一可能之源，是纯质于自身同一性中的内在极化。此极化非实体之裂变（那将破坏同一），而是功能性的自反性构型——即“显能”与“显项”在原初同一中的功能性分化。此一分化，是“秩序”得以从“同一”中诞生的唯一逻辑端绪，它先于且使得任何后天的主—客二分得以可能。

至此，“必然自动”已立。动既必然，其所能是否有限、其如何自为地生成结构，将成为之后论证的任务。
