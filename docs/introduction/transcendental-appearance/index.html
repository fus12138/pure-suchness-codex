§0 显现之先验位阶确立

显现
指任何事物得以呈在、得以有其位、得以成为“可在之者”之根本事件。
它不等同于被意识知觉，不等同于对象进入表象，更不依赖语言、概念或主体。
显现先于主体与客体之分，是一切“能-所”关系、认知结构与逻辑框架得以成立之前的自明之敞开。

世界得以为世界，而非隐没于无法言述之域，因其显。
显现不是某物之属性，而是“有所为物”成为可能的第一条件。
它不是认知之结果，而是认知得以可能之先在地。

§0.1 哲学追问的零度立场

哲学之思，并非始于对世界之确认，亦非始于主体之自知，而始于更原初且无法逃避的事实：有物显现。
非因心而后显，非因认知而成其为显。

在意识之前，现象已在场；在概念之前，世界已敞开。
一切“存在”之谈，一切“认识”之问，仅能在已有显现之域内展开。

然而，此“已显”常被无条件、无反思地作为前提承接，而极少成为哲学思索的对象。
于是一个根本之问浮现：显现本身，如何可能？

此问不诉诸经验，不依赖主体的存在，不以先在世界为凭，更不假设逻辑作为外部立法者。
它只要求思意识到：凡可言、可问、可思，皆已被显。

故在理性思索中：
“显现如何可能”先于“何物存在”与“如何认识”。

前二者以显为起点，此问则探向起点自身。

§0.2 显现之不可回避性

存在论之命题若欲成立，必须承认“存在者得以显”。
若无显现，则“存在”不入视域、不可被言。

认识论之论证若欲展开，必须承认“能知与所知”已处于同一明域。
若无显现，则对象与主体均不可被定性为“认识之项”。

故显现之问非从属于存在论或认识论，而是二者之共同前提。

所谓“能显”与“所显”，并非显现的先条件，而是显现内部逐步分化的结果。

若反向将此分化当作显现之基，逻辑必陷于无限后退：
需先解释能与所之显，又需解释更前之显，无穷无尽，无立足点。

因而显现之地位为：
不可削减、不可转移、不可后退。

该处即哲学之零度底面。

§0.3 先验清理与边界划定

显现并不以主体—客体结构为前设；
“能显—所显”是显后之分野，不是显之条件。

显现亦不以认识范畴为根基；
主体、客体、对象、意向性等分类，皆属显内之结构，非显外之法。

更进一步，逻辑律（同一、不矛盾、推演）亦非显现之外的绝对尺度，
而是显现过程中所展露之内在秩序。

此等整理，并非削弱既有哲学，而是将所有体系安回其位：
它们得以成立，因为显已在前。

§0.4 纯粹起点与方法自律

由此，本论之起点须立于：

只承认显现之事实，不预设任何显中之事物。

不得携入既定主体，
不得偷引既存世界，
不得引入概念、范畴、时空或因果作为前提。

此处唯一许可之问题是：
显现作为显现，其可能性何在？

为防思误入旧轨，需守三律：

  1. 不引二元：显非“为某者显”之事件。
  2. 区分先验与动力：显之可能性 ≠ 显之实现过程。
  3. 逻辑内生：逻辑不裁定显，逻辑在显中得其权威。

因此，本论并非立存在论，亦非建认识论，
而是为二者奠其可成立之地。

如筑屋者不先论室内之器，而先求地基之固。
显现之问，即哲学之奠基之问。

立此，则思能起；
失此，则思无地。
