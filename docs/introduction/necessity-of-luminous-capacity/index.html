§1 “使显之能”的先验必然性

显现既立为不可后退之原点，理性不容回避的诘问随之升起：显现既已事实地成立，其可能性 根据何在？
此问，乃理性对自身尊严的守护。
拒斥它，意味着容忍理性在起点崩塌；
接纳它，思想才真正踏上为其奠基的征途。

§1.1 理性之责与追问之必然

任何事实的成立，都必然召唤对其可能性的追问。“显现如何可能？”——此非随意之问，乃先验思辨的绝对必然。
对此问题的态度，划分了思想的疆域：
拒之，则堕入非理性的奇迹；
承之，则步入哲学的真正堂奥，为显本身寻获其所以能显之终极根据。

§1.2 显现为成就，故有其条件

显现，绝非静态的“已有”，而是从“不可及”跃入“可及”之域的根本成就。
凡成就，必预设其可能性条件。无条件之成就，无异于纯粹的偶然，将使理性无处立足，一切论证失去根基。
因此，追问显现之条件，是理性的自我救赎，是思想得以成立的第一命令。

§1.3 可能性条件的穷尽排除与必然归结

理性须以逻辑为刃，对一切候选进行彻底的先验清洗：

1.非所显之物： 一切存在者皆已是显现之结果，以果为因，则陷入无限回溯。
2.非空无： 空无不能作为任何成就的依据，“无”不能生“明”。
3.非逻辑律： 逻辑本身需要显现的场域来展开其有效性，故不能先于此场域而为其立法。
4.非主体或意识： 此二者均为显现内部后天构成的结构，是“显后之果”，非“显前之因”。
至此，所有歧路皆被堵死。思被逼至绝境：若不在此立一“能”，则显现本身将沦为不可思之谜。

§1.4 明性之必然：理性所迫之立基

逻辑的穷尽处，正是思想的真正起点。理性别无选择，必须确立：
显现必以一“非对象化的、自明的、自我奠基的‘使显之能’”为其先验条件。
此即明性。 （将在下一节详细论述）
它非被给予，而是使一切给予成为可能；
它非被照见，而是使一切照见得以发生。

§1.5 反证之刃：若无明性则思亡

反设：无此明性。
则：
显现无因 → 成就无根
经验无地 → 对象无域
理性无基 → 论证无始
逻辑无凭 → 法则失效
世界不敞 → 归于寂暗
最终，语言坍塌，思想断流，哲学在开端处自我毁灭。
故，立此明性，非为构造神学，乃为护理性之尊严，守护思想与世界的可能性。

拒之，则万古如长夜；
立之，则世界得以为世界。
