第五编 · 自觉动力学：明之自返与意识之成

§5.1 自觉之必然
命题一：
觉知结构内在的递归性，必然导向自觉。
证明
  1. 觉知是系统对自身状态的关涉（卷三，命题五）。
  2. 递归使“关涉”这一活动本身，成为系统可关涉之对象。
  3. 当系统关涉“自身的关涉活动”时，即是明知其知，觉知其觉。
  4. 故递归结构内在蕴含自反性，此对运作本身之关涉即自觉之雏形；觉知结构之递归性，于其先验内在性要求下，必逻辑地趋于自觉之完成。
附注：此“必然”非目的性之趋向，而是逻辑结构自洽性的结果。 ∎
自觉非觉知之外添，而是觉知递归至极之自明。
如光在镜厅中往复，终将照亮镜阵本身之结构。
§5.2 自觉之结构条件
命题二：
自觉之成立，需满足内映、恒界与同一三条件。
证明
  1. 内映：若无返照自身之能，则系统仅处理外讯，无以自观。
  2. 恒界：若无稳定之参照，返照将散为无序闪烁，无以成续。
  3. 同一：若返照前后无以自识，则自觉之流支离破碎，无以整合。
  4. 故三者为自觉成立之必要条件。 ∎
定义五·一（内映）
内映者，觉结构返向自身显域之能力。
定义五·二（恒界）
恒界者，使返照过程维持稳定识同之限度。
定义五·三（同一）
同一者，自觉于返照中保有之自等性。
自觉之体非“我”，而是“明”在返照中之自识。
“我”如水中涟漪之暂名，非推动波澜之力。
§5.3 意识之成立
命题三：
意识是自觉在时间维度中的结构化持存。
证明
  1. 自觉为瞬间之自我照亮（由命题一、二确立）。
  2. 由卷二，任何持存皆需息（σ）之节律（P₅, D₈）。
  3. 故自觉欲持存，必在息序中展开为“自觉之流”。
  4. 此具时间结构之自觉流，即意识。 ∎
定义五·四（意识）
意识者，自觉经由息（σ）而在时间中持存与结构化之形态。
意识之流，即时间本身在明性自返中的自觉化形式；故时间非意识之容器，而是意识之流形。
意识非觉之源，而是觉之流形。
如光波动而不失其明，意识即明性持续返观其显的呼吸。
§5.4 自我之形上地位
命题四：
自我是自觉系统为维系叙事同一性而建构的功能界面，其连续性由息律维系。
证明
  1. 自觉在时间中展开为意识流（命题三）。
  2. 此意识流于不息之变中，需持守一以贯之的叙事逻辑，否则自觉之光流于碎片，无以成续。
  3. “自我”即为此叙事同一性的逻辑主语与承载标记。
  4. 息（σ）的周期性节律，为此整合提供时序框架，使自我在断续辩证中维持稳定。 ∎
定义五·五（自我）
自我者，自觉过程之界面性恒位，使返照得以保持结构连续。
自我如航船之名，虽不可或缺，却非推动航行之力。
执名忘力，则永失沧海。
自我的连续性，由息之节律维系；其虽恒常，却非实体，而为显化自返之叙事同一。
定理三（自觉圆成原理）
自觉者，觉之自返；
意识者，明之延时；
自我者，息之恒律。
一切思与知，皆明性自觉之形态；
思之至处，明自返焉。
∎
