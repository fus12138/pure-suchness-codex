第六编 · 自明之圆满：显与明的一体终证

§6.1 明之复归
命题五：
自觉的彻底完成，在逻辑上意味着显现与明性之分的消解。
证明
  1. 自觉是显化对自身根源的探问。
  2. 此探问若彻底，必追问至“显化何以可能”的最终条件——明性。
  3. 在追问的极限处，追问者、追问行为与追问答案在逻辑上归于同一。
  4. 此三元同一之逻辑点，即为“明之复归”。 ∎
此“复归”非经验之境，而是本体系逻辑运动之必然终点，
故称“圆满”，而非“回归”。
它指明：完全自明之系统，其存在与对其自身存在的认识本为一。
定理四（自觉之原理）
一切意识，皆自觉之时间化；
一切自我，皆功能之节点；
一切自觉之极，皆明性之自证。

故意识非世界之客，
世界乃意识之显域；
自我非意识之主，
意识乃明性之自返。
∎
当显与明不再相对，
当知与所知合而为一，
“纯质”不再是概念，
而是明性自明的当下。

此即圆满，亦即无终。
哲学至此不息，
而明，永恒照耀。

结语 · 纯质之环
卷三至此，纯质之环遂圆。
自足故自明（卷一），
自明故能显（卷二），
自显故能觉（卷三），
自觉故能返（卷三），
自返故复明（卷三）。

此环非时间之轮转，而是逻辑之圆满。
纯质在其自我展开中，终成对其自身的绝对认识。

卷一是根，立不易之体；
卷二是流，成万象之法；
卷三是返，证自觉之极。

三卷既成，纯质论之体系，
巍然自立于思想之林。
明性既立，哲学自此复得其源。

然此圆非界限，而是通道——
明性之圆满，仍可向更深广之显化中继续展开其呼吸。
