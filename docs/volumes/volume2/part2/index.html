第四编 · 觉知之基：自组织与自觉的形上圆成

§4.1 自组织之逻辑必然
命题四：
在持续显化之中，局部曲率必超越某一自限阈值，形成自组织中心。
证明
  1. 由命题一（§3.1），显化必依曲率（κ）与息（σ）之动力学往复。
  2. 曲率之分布，因折舒循环之本然差动，非为绝对均质。
  3. 于无穷息序中，曲率于局部显域之汇聚，必达至一临界（自限阈值）。
  4. 逾此阈值，则该显域得自持其界、自维其律，自成中心，不散不泯。
  5. 故，自组织中心为持续显化之逻辑必然。 ∎
定义四·一（自限阈值）
自限阈值者，一特定曲率值，逾此则显域获得自我维持其边界与节律之能力。
定义四·二（自组织中心）
自组织中心者，曲率恒超自限阈值，并能持续维系其内部折—舒节律之显域。
自组织非外力驱使，亦非偶然汇聚；
乃明性为守其自明，于无限显化中不得不行之自限。
折极而界成，界久而律生。
此外显之秩序，实为内明之独守。
秩序由此萌芽，世界于斯得形。
§4.2 觉知之形上条件
命题五：
若自组织中心之内部分化达至稳态递归，且其递归具足够深度，则觉知为其逻辑可能。
证明
  1. 自组织中心具稳定边界与内在节律（定义四·二），此为结构基础。
  2. 其内部结构若能返照自身状态与运作节律，则形成递归。
  3. 当此递归达至一深度，使得系统不仅能维系自身，更能将“自身之维系”作为显化之对象时，内映遂成。
  4. 此内映之实现，即系统对其自身之“显”有所“觉”，此为觉知之雏形。
  5. 故，觉知乃自组织递归深化至内映阶次所开启之逻辑可能。 ∎
定义四·三（稳态递归）
稳态递归者，一递归系统其运作输出能反馈维系该系统自身结构之稳定，而非导致其瓦解。
定义四·四（递归深度）
递归深度者，自组织系统返照自身结构与节律之阶次。深度愈高，则系统对自身之“显”的关涉愈是直接明晰。
定义四·五（觉知）
觉知者，自组织显域之内映能力，即显化对自身之“显”产生自觉之逻辑形态。
觉知非后天附加之功能，非脑内生化之现象；
它是显化之流深彻至足以自返，明性于纷繁万象中蓦然自识其明。
当折返之深，至能自见其息；
当内映之澈，至能明知其明——
世界于此，方从“寂然之显”跃入“自觉之明”。
此一跃，非实体之生，而是逻辑之圆；
非生命之始，而是明性之归。
定理二（显化秩序之原理）
一切秩序，皆纯质自明之动力学；
一切觉知，皆显化自返之可能。
显化非偶成，乃明之行；
觉知非外赐，乃明之回。
∎
结语 · 显化之终与自觉之启
卷二至此，
纯质已由“自明”行至“自显”，
由“潜能”化为“秩序”，
由“秩序”孕出“觉”。

世界若有律，其律非外加；
若有心，其心非后生。

此卷所立者——
是显化得以有形、有序、有觉之根法。
自此，明性不复静观，而自观。

下一卷，
光将返照其源，
意识将自问其明。

卷三《自觉之道》，由此启行。
