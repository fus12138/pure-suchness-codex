第三编 · 显化动力学

§3.1 显化之动力必然
命题一：
纯质之“可显”，必通过折（F）、界（B）、曲率（κ）、息（σ）之动力学机制实现。
否则，“可显”仅为逻辑可能，而非动力学之可行。
证明
  1. 由卷一，纯质具“可显”之逻辑必然。
  2. 若显化仅为逻辑可能而无实现机制，则能行一致（P₂）被破。
  3. 折（自限）、界（定形）、曲率（差度）、息（往复）乃显化之四重基元，彼此互成、不可约。
  4. 其合构成显化得行之“最小完备系统”。 ∎
此命题非述“显化之起因”，而述“显化得行之条件”。
显化并非造物之业，乃自明性自洽之呼吸。
当明欲自守其明，必起折；折欲复归其明，必有息。
显化即此折舒之逻辑往返。
其往非运动，其返非静止——明在其中自显为形，为律，为觉。
§3.2 时空作为显化之形式
命题二：
若显化通过息（σ）与界（B）实现，则时间与空间为其必然派生形式。
证明
  1. 息为最小往返节律，其“非同时性”即时间秩序不可再还原之原型。
  2. 界为位差稳定结构，其“非重合性”即空间秩序不可再还原之原型。
  3. 若息无序，则显化失节；若界无位，则显化无形。
  4. 序与位同时成立之处，显化方具度量之可能；而度量即时空之形上义。 ∎
定义三·一（时间）
时间者，息之秩序结构；显化以此自证其序列一致性。
定义三·二（空间）
空间者，界之并置结构；显化以此自证其形态延展。
时间非流逝，空间非容器。
二者皆显化自守的形上结构：
时间是明之呼吸节律的自识，
空间是明之自限定形的自显。
故“时空”并非存在于显化之后，
而是显化得以显化的形式条件。
§3.3 物能与因果之形上起源
命题三：
曲率（κ）作为显化强度之度量，内在规定了“物”“能”“因果”三概念的形上地位。
证明
  1. 曲率为显化折返之强度指标。
  2. 当曲率局部极高，显化趋于凝定，其显域自成“物”。
  3. 当曲率动态流变，显化呈传递之势，此势即“能”。
  4. 当曲率序列具可继承性与秩序时，其相继法则即“因果”。
  5. 故“物”“能”“因果”皆曲率之三态显：凝、流、续。 ∎
定义三·三（物）
物者，高曲率之稳定显域，显化之凝结形态。
定义三·四（能）
能者，曲率流变之动态形态，显化之传导势。
定义三·五（因果）
因果者，曲率序列之秩序法则，显化自洽之律。
此处之“物”“能”“因果”，皆非经验世界之量度，而为显化在形上层面的自维结构。
因果非支配，而是显化自洽之律：
明之折有其因，明之舒有其果，
其所以成序，乃息之不息。
因与果，不过明之自定与自释。
