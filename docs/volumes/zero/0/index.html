显现之先验位阶之确立

§0.1 哲学追问的重新定位
条文一：哲学之第零问题
一切哲学之思，皆始于"某物显现"这一基本事实。然此事实本身未被思及，却已成为存在论与认识论之默认前提。
故，在问"何物存在"或"如何认识"之前，须先问：
"显现本身，何以可能？"
此问不预设世界，不援引经验，不依赖主体，唯追究"使显现成其为显现"之先验条件。
条文二：存在论之预设
存在论以"存在者之存在"为课题，然其已预设：存在者得以显现为存在。
若无显现，则"存在"无从被思、被言、被追问。
故存在论立于显现之基，却未思此基。
条文三：认识论之预设
认识论以"认识如何可能"为出发点，然其已预设：认识对象与认识主体皆已显现于某种场域之中。
认识是显现之一种形态，而非显现之根源。
故认识论研究显现之内部结构，却未追问显现本身之可能性。
§0.2 显现之不可退性
命题一：
显现问题，在逻辑上先于存在论与认识论。
证明
  1. 存在论与认识论皆以"某物已显"为出发点。
  2. 若"显现"本身不可能，则存在与认识皆无从谈起。
  3. 故，追问"显现如何可能"，在逻辑上优先于追问"何物存在"或"如何认识"。 ∎
命题二：
显现问题，非存在论之一部分，亦非认识论之一分支，而是二者共同之先验前提。
证明
  1. 存在论处理"所显之在"，认识论处理"能显之识"。
  2. 然"所显"与"能显"皆在显现之中分立。
  3. 故显现问题，乃是"能—所"得以分立的可能性条件，居于二者之先。 ∎
§0.3 对潜在诘难之先验回应
命题三：显现非二元
显现本身不预设"某物"与"能显"之二元分立。二者乃显现内部之功能位格，而非显现之先验条件。
证明
  1. 若显现预设二元，则此二元本身需被显现，导致无限后退。
  2. 故显现必为自足之事件，其内部之能所分化乃后续衍生之事。
  3. 此自足性即明性之自呈性，为显现可能之终极条件。 ∎
命题四：显现非认识结构
显现虽以概念被指称，然其本身先于一切认识论结构。
证明
  1. 认识结构（如主体、客体、范畴）皆已预设显现。
  2. 故显现非认识论之一环，而为认识论之基础。
  3. 以概念指称显现，乃不得已之权便，须防概念之实体化。 ∎
命题五：逻辑不先于显现
逻辑形式本身亦是一种显现，而非显现之前提。
证明
  1. 逻辑规则（如同一律、不矛盾律）皆在思维中被显现。
  2. 若逻辑先于显现，则逻辑本身如何显现？其必然陷入预设之循环。
  3. 故逻辑乃显现之内部规律，而非显现之外在条件。 ∎
§0.4 纯粹逻辑之出发点
定义一：显现
显现：指任一"在者"成为"可被指向者"之基本事件。不预设主体、客体、时空或因果，仅指"从隐至显"之逻辑可能性。
公设一：非预设性起点
哲学之先验探询，不得预设任何已显之存在者或认识结构。唯能从"显现如可能"此一纯形式问题出发。
方法论警示
  1. 防二元渗漏：凡将"显现"理解为"某物之显"或"对某之显"者，皆已暗渡能所二元。
  2. 范畴边界："显现如何可能"属先验可能性问题；"显现必然现实化"属动力学实现问题。二者须严格区分。
  3. 逻辑自反：本论所用之"逻辑"，非外在于显现之工具，而是明性自呈之有序形态。逻辑之显现，将在后文专论。
附释
本论不回答"何物存在"，亦不建构"认识之法则"，而是为二者奠定不可动摇之基。
如筑屋者不先问"屋内何物"，而先问"屋何以能立"。
此即哲学之第零问题，亦为纯质论之真正起点。
一切概念皆权便，一切逻辑皆内显。
唯此一问，直指源头。
